## What Art Protocol Studio Is (Revised Vision — 19:30 IST)
**NOT an agency. A self-serve AI brand studio for founders and entrepreneurs.**

Core loop: Sign up → Create project → Fill brief (conversational) → Run AI crews → Get beautiful formatted outputs → Edit via chat → Export/Share → Deploy

Key differentiators:
- **Memory stack**: every crew reads what ran before (research informs branding, branding informs social, etc.)
- **Not just text files**: rendered documents, color swatches, formatted deliverables
- **Chat-to-edit**: every output is a live document, chat to tweak/regenerate
- **Versioned outputs**: full history
- **Self-serve**: users run their own crews, charged per run via credits
- **File upload path**: user can skip research, upload their own brand doc, run social/ads directly
- **Future**: Shopify integration, ad platform APIs, MCP deployment
